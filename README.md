# The Neon Passage

The Neon Passage is a real time 3D environment project made in Unreal Engine.

This project focuses on lighting, materials, and atmosphere. The main idea was to create a neon tunnel scene that looks strong visually while still running well in real time. :contentReference[oaicite:0]{index=0}

## Project Overview

The Neon Passage is a futuristic environment with neon lighting, glass surfaces, and an outdoor natural setting. The aim was to build a scene that feels calm, atmospheric, and visually interesting. The project also explored how artistic choices and technical work come together in a game engine. :contentReference[oaicite:1]{index=1} :contentReference[oaicite:2]{index=2}

## Main Focus

This project was mainly about learning how rendering techniques can improve the look of a game environment. I worked with materials, lighting, post processing, and optimisation to build a scene that looks good and performs well. :contentReference[oaicite:3]{index=3} :contentReference[oaicite:4]{index=4}

## What I Made

I created a neon tunnel environment with glowing lights, glass panels, and an outdoor area around it. The scene was designed so the player moves through the tunnel while looking out into the environment. This helped keep the level simple while still making the world feel larger. :contentReference[oaicite:5]{index=5} :contentReference[oaicite:6]{index=6}

## Challenges

One challenge was fixing stretched textures caused by UV mapping problems. Another was balancing the glass, reflections, and neon lighting so the scene stayed clear and readable. I also tested baked lighting, but it caused performance problems, so I used simpler methods that worked better for this project. :contentReference[oaicite:12]{index=12} :contentReference[oaicite:13]{index=13} :contentReference[oaicite:14]{index=14}

## Future Improvements

If I continued this project, I would improve the textures, learn baked lighting properly, optimise more of the outdoor assets, and add more detail to the environment. I would also refine the balance between the neon tunnel and the outside world. :contentReference[oaicite:15]{index=15} :contentReference[oaicite:16]{index=16}

## Summary

The Neon Passage helped me learn how environment art, lighting, and optimisation work together in real time. It gave me a better understanding of how to build a scene that looks good, feels clear to the player, and performs well in engine.
